# عواصف شمسية وصمت راديوي في المنطقة العربية
## دراسة حالة لتوهج شمسي قوي – 14 مايو 2025

يحتوي هذا المستودع على جميع المواد العلمية المصاحبة للدراسة:

**"عواصف شمسية وصمت راديوي في المنطقة العربية:  
دراسة حالة لتوهج شمسي قوي في 14 مايو 2025"**

بما يشمل:
- الأشكال المستخدمة في المخطوطة العلمية
- الفيديوهات التكميلية (Supplementary Movies)
- دفاتر Jupyter المستخدمة في التحليل وإعادة إنتاج النتائج
- الأكواد البرمجية لمعالجة بيانات GOES وDRAP
- البيانات المعالجة ضمن حدود المنطقة العربية

---

## محتويات المستودع

- `figures/`  
  جميع الأشكال النهائية وعالية الدقة المستخدمة في الورقة.

- `videos/`  
  تطور التوهج الشمسي (SDO/AIA) والتغير الزمني لامتصاص HF (DRAP).

- `notebooks/`  
  دفاتر تحليل البيانات وإعادة إنتاج الرسوم.

- `data/`  
  بيانات خام ومعالجة (GOES X-ray flux و DRAP maps).

- `scripts/`  
  سكربتات Python المساعدة في التحميل والمعالجة.
